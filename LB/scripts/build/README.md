# build
Collection of Loadbalancer.org scripts for system building.

README.md - This file.

cloud-tools.sh - Install cloud tools for Rocky 8 build system.

configure.sh - Perform initial configuration of VA for testing.

Deploy.sh - Performs required steps for, and including, git deployment.

update-Deploy.sh - Update Deploy.sh.

update-update-Deploy.sh - Update update-Deploy.sh.

MakeRPMs.sh - Just build RPMs for deployment.

MakeImage.sh - Run steps for release images.

finish.sh - Perform post-deploy steps for release images.
